Features

Project OP2 currently implements the following features:

  • HD audio calls
  • Multiple SIP accounts
  • Speed Dialing
  • Web configuration interface
  • Built with SIP SIMPLE SDK
  • REST API to control everything
Since it's based on SIP SIMPLE SDK, more features available in the SDK wil be made available in OP2 in the future.

Download

Here are all the packages you need to get Project OP2 up and running. We are not hosting all required packages, only those which are not present in Raspbian.

  • op2d (main daemon)
  • python-sipsimple (special edition with some extra patches)
  • python-sipsimple-dbg (debugging symbols)
  • python-application (version 1.4.0)
  • python-eventlib
  • python-msrplib
  • python-xcaplib
